# Calm Mahjong

A relaxing Mahjong-style tile matching puzzle game built with HTML, CSS and JavaScript.

## Features

- Relaxing Mahjong-inspired gameplay
- Clean and simple design
- Mobile friendly layout
- Smooth tile matching mechanics
- Lightweight and fast

## Tech Stack

- HTML
- CSS
- JavaScript
